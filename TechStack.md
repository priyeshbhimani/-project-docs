# Tech Stack
  Frontend: React 18 + Vite | Tailwind CSS | shadcn/ui | Framer Motion | Lenis
  Editor: Fabric.js v6 | State: Zustand | Forms: React Hook Form + Zod
  Animation: lottie-react | Icons: Lucide React
  Backend: Node.js v20 + Express.js | Auth: Supabase Auth + JWT
  Security: Helmet + CORS + express-rate-limit | Validation: Zod
  AI: OpenAI DALL-E 3 (primary) + Stability AI (fallback) + LottieFiles AI
  Storage: Cloudinary | DB: Supabase PostgreSQL | Cache: Upstash Redis
  Payments: Stripe (intl) + Razorpay (India) | Email: Resend
  Deploy: Vercel (frontend) + Railway (backend) + Cloudflare (DNS+CDN+WAF)
  Monitor: Sentry (errors) + PostHog (analytics)
